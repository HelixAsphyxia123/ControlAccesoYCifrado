# Programa control acceso con cifrado SHA-256 usando Java.
 
 *Se desarrollo un programa que usa el formato MVC (Modelo, Vista y Controlador) con el fin de registrar usuarios, dandoles la oportunidad de registrar un nuevo usuario para pasar a la siguiente ventana, en caso de que este no se encuentre registrado, se le denegara el acceso y se le volvera a preguntar la contrase침a.*
 
 *Se uso un cifrado en la contrase침a llamado SHA-256 (sugerido por ser mas reciente en version).

 # *Elementos visuales y funcionamiento.*
 *~Dos textLabel que nos muestran en que parte hay que ingresar nuestras credenciales(password y contrase침a).*
 
 *~Dos JButton, uno para Entrar dependiendo de las credenciales ingresadas, y el otro para registrar Usuarios.*
 
 *~Un JPassword para ocultar nuestra contrase침a.*
 
 *~Un JTextField para ingresar el usuario.*
